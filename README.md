1. Что такое калькулятор?
------------------------
Калькуля́тор — электронное вычислительное устройство для выполнения операций над числами или алгебраическими формулами.
Калькулятор используют для того, чтобы считать цифры.
Калькуляторы бывают:
* Простейшие;
* Инженерные;
* Бухгалтерские;
* Финансовые;
* Программируемые.
